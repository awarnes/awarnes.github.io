The 5th Edition Character Sheet Manager is a pet project whose aim is to take the pain out of Dungeons & Dragons, while still affording the player all of the creativity that the game is based on.

The 5E CSM is built in Django with a bed of Python, and plenty of JavaScript and JQuery to interact with the database through a built-in REST-ful API.

Check out the code on github here !

Languages and Features: Django, Python, JS, JQ, RESTful API Date: April 2017 Service: Web Development

TwitterMaze

PreviousNext

Combining the old and the new, TwitterMaze invites players to navigate the twists and turns of the most popular tweets in the USA!

Created in concert with Danny Burrow, the tweet in question is pulled live from a randomly generated list. It is then embedded into a maze using a homespun algorithm. Finally, players are able to navigate through the maze in a Curses based front-end for the Terminal.

Check out the code on github here !

Languages and Features: Python, Tweepy, Curses, Twitter API Date: February 2017 Service: CLI Development and API Integration

Portland Bus Stops

PreviousNext

Finding your next bus is always important, but what if you could see exactly where a bus was and how long it was going to take to get to your stop, so you didn't have to step out into the rain until absolutely necessary?

The focus of the bus stop finder is to help people get information about their next bus based on their geo-location. Once they determine which bus they're looking for a live update screen shows exactly where each bus on that route is updated every 5 seconds from the TriMet API so that you never have to wait in the rain again (though it is Portland, so where's the fun in that?).

Check out the code on github here !

Languages and Features: JavaScript, GoogleMaps API, TriMet API Date: February 2017 Services: API Integration and Geo-Location Services
